Trackword Search

This search mode allows to find words in the grid of letters. The program moves from letter to letter horizontally, vertically or diagonally. Each letter can be used once or few times for the word creating.

Grid dimensions can be from 3 x 3 to 30 x 30. User can search either in specified length word list or in all database. Also search directions can be defined by user.

User can view found words in the trackword. Click the right button of mouse on the word in the search results window and choose the menu item Show Track.
