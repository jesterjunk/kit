Coupling Words Search

This search mode allows to find pairs of words, in which the first word ending is equal to the second word beginning. User defines word lengths and amount of common letters. The character ? is used instead of unknown letter, coincident letters should be written in round brackets. User can define, whether each part of pattern should be a word from Cross+A database or not.

For example: ????(book)???? = note(book)mark, notebook and bookmark.
