Using Search Modes

Choose the main menu item or press the button of the search mode you need. For example, if you want to search by pattern, choose the main menu item Search|Search by Pattern or press the button .

User can use the main menu item Edit|GoTo for the fast navigation in the database. Also, user can press CTRL and click on any word in description; the program will go to this word and select it.

For some languages a letter with the diacritic mark can be written without it. Use the main menu item View|Equivalent Letters to list such possible replacements.
