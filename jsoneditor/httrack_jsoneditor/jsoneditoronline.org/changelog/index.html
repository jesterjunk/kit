Changelog

A detailed overview of the changes of every release of https://jsoneditoronline.org.

Releases

2025-04-15, version 7.9.0

  • Feat: introduce monthly and annual pricing plans alongside the quarterly plans, see https://jsoneditoronline.org/pricing/.

2025-03-28, version 7.8.0

  • Feat: truncate large strings in tree and table mode.
  • Feat: precisely scroll to search results inside a large text in tree mode.
  • Implement quickkeys Command+S and Command+O to save and load files on Mac.

2025-03-11, version 7.7.1

  • Feat: update dependencies.

2025-02-14, version 7.7.0

  • Feat: pasting in via the context menu in tree and table mode is now supported.
  • Feat: improved jsonrepair, now repairing fenced Markdown code blocks.

2025-01-29, version 7.6.0

  • Feat: implemented new hash parameters left_mode, right_mode, panels, and compare, allowing to control the mode and visibility of the panels when loading documents via the left and right hash parameters. See docs: https://jsoneditoronline.org/docs/#url.
  • Updated dependencies.

2025-01-17, version 7.5.3

  • Fix local legacy documents not being migrated when the migration has been executed once before.

2024-12-05, version 7.5.2

  • Fix modals closing when the mouse ends outside the modal after selecting text.

2024-12-03, version 7.5.1

  • Fix JSON schema validation not working anymore. Regression introduced in v7.4.0.

2024-12-03, version 7.5.0

  • Update to @jsonquerylang/jsonquery@4.1.1, containing new functions reverse, mapObject, mapKeys, mapValues, join, split, substring, number, and string.
