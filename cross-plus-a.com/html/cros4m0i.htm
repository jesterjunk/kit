Word-chain Search

This search mode allows to find chains of words, where every word differs from its neighbors by one or few letters. This type of puzzles is also known as "Doublets", "Laddergrams", "Ladderwords", "Word Ladders", "Stepwords", "Transitions", "Transformations" etc. First word, last word and amount of changeable letters are specified by user. Also anagrams can be used in word-chain creating.

There are five variants of this search mode:

  • Change letter.

    For example: fist - fast - fact.

  • Change letter and/or use anagram.

    For example: soft - fort - form.

  • Change letter and use anagram.

    For example: fist - soft - fort.

  • Change letter or use anagram.

    For example: post - stop - shop.

  • Find neighbours for the word.

    Program finds words differing by single letter from the pattern. For example, the list of neighbours for the word start: smart, stare, stark, swart.
