Search by Description

This search mode is used when some words in the target word description are known. User can search either in specified length word list or in all database. The known words in description are enumerated via blanks.

For example: length unit metric

The special wildcard characters can be used in search query:

  • ? is used instead of one unknown letter;
  • * is used instead of few letters;
  • @ is used instead of any vowel;
  • $ is used instead of any consonant letter;
  • & is used instead of few vowels;
  • # is used instead of few consonant letters.

The character ! changes the meaning of next symbol on opposite. If there are few possible letters on the specific position in the word, these letters must be written in round brackets.

For example: len??? unit m*c

To query on symbols that have special meaning to query expressions such as ? or round brackets, you must escape them. Use the backslash character to escape a single character or symbol. Only the character immediately following the backslash is escaped. For example:

   AT\&T

   \(comment\)

"Weight" of found words also can be considered in search process. User can define precise the "weight" value for word searching. Also the program can underline found words in descriptions at the window of search results (check box Show found words in descriptions).
