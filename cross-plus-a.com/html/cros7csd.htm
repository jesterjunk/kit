Consecutive Sudoku

In Consecutive Sudoku ("1-away Disallowed Number Place"), all the places where orthogonally adjacent cells are consecutive numbers have been specially marked.


In Non-Consecutive Sudoku no two adjacent cells have consecutive values. For example, if a cell contains 3, any neighbouring square cannot contain either 2 or 4.

The formats of sudoku files:

  • CSD – Consecutive Sudoku (9 x 9);
  • CSD6 – Consecutive Sudoku (6 x 6);
  • CSD8 – Consecutive Sudoku (8 x 8).
